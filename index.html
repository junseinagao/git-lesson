Gitって何?

Gitは、ソースコードを管理するためのツール

Gitは、ソースコードの変更した履歴を保存することができる。

ポケモンだと、「レポート」(セーブデータ)を書いて進行状況を保存する

Gitもソースコードのセーブデータを作れる

Git レポジトリ

git init でGit レポジトリを作成する

git add & git commit で変更履歴を保存する

git add と git commit

gitレポジトリに履歴を保存することをコミットという。

Gitには3つの段階がある

  • 今編集している段階
  • これから、コミットするよーってファイルを選ばれた段階(ステージング)
  • コミットされましたという段階

git addは、ステージングするファイルを選んでいる

git add . は、ファイルを全部ステージングする

git commit でコミット(セーブ)する

ブランチ

セーブデータを複数個作ることができる

git branch ブランチ名 でブランチを作れる

git checkout ブランチ名 でブランチを移動できる

Git マージ

ブランチを取り込むことができる

Github

Gitのレポジトリをアップロードしておけるサイト

Gitレポジトリ(セーブデータ)をクラウドに保存しておけるみたいな感じ

このGitレポジトリをGithubにアップロードする
